! mgexecutor v42.0

Продвинутая среда исполнения JavaScript для вселенной Rublox

Что такое mgexecutor?

Это полноценный внутриигровой инструмент автоматизации. В отличие от простых читов, он предоставляет вам настоящую среду разработки (IDE) на языке JavaScript прямо внутри запущенной игры.

Вы можете писать собственные JS-скрипты так же, как разработчики в официальной студии Rublox, перехватывать потоки данных и управлять действиями своего персонажа: активировать полёт, телепортироваться, настраивать супер-скорость или подсвечивать объекты.

Пошаговая инструкция по установке

Шаг 1. Включение режима разработчика

Чтобы браузер разрешил установку сторонних скриптов, откройте настройки браузера -> Расширения и в правом верхнем углу включите тумблер «Режим разработчика».

Шаг 2. Установка менеджера скриптов

Установите официальное расширение Tampermonkey для вашего браузера. Если не знаете как, посмотрите подробную инструкцию в сети.

Шаг 3. Добавление mgexecutor
  1. Нажмите на зеленую кнопку скачивания выше, чтобы загрузить файл mgexecutor-42.0.js.
  2. Откройте скачанный файл через любой текстовый редактор (например, Блокнот) и полностью скопируйте всё его содержимое (Ctrl+A -> Ctrl+C).
  3. Нажмите на иконку Tampermonkey в панели расширений браузера и выберите «Создать новый скрипт...».
  4. Полностью сотрите весь базовый текст в открывшемся редакторе, вставьте скопированный код экзекутора и нажмите Файл -> Сохранить.
Шаг 4. Запуск в игре

Перезапустите вкладку с игрой Rublox. После входа на игровую карту нажмите на клавиатуре клавишу Insert или ~ (Ё). На экране появится стильный неоновый интерфейс экзекутора!

⚠️ Важно: Скачивайте обновления только с этой официальной страницы, чтобы гарантировать безопасность ваших личных данных и игрового аккаунта.

База скриптов (Вводить в консоль)

Скопируйте любой из этих скриптов, вставьте в текстовое поле mgexecutor в игре и нажмите кнопку EXECUTE TO WORKERS.

1. FlyHack & UFO (Полёт)

Отключает стандартную гравитацию для персонажа и позволяет ему летать во все стороны.
// Активируем режим НЛО game.player.setUfoMode(true); game.player.setShipMode(false); game.player.setSpeed(30); console.log("[Hack] Режим полета активирован! Управление на W,A,S,D + Пробел/Crouch");

2. SpeedHack (Супер-скорость + Мега-прыжок)

Разгоняет персонажа до безумных скоростей, увеличивает силу прыжка и активирует «двойной прыжок».
game.player.setSpeed(60); game.player.setJumpPower(35); game.player.setDoubleJump(true); game.player.setIceFriction(0); console.log("[Hack] SpeedHack и JumpHack включены!");

3. Noclip (Прохождение сквозь стены)

Заставляет физический движок думать, что игрок может проходить сквозь любые объекты.
if (typeof physics !== 'undefined' && physics.passThrough) { physics.passThrough(true); console.log("[Hack] Noclip активирован. Вы можете ходить сквозь стены!"); } else { game.every(0.1, function() { if (game.player.isKeyDown('w')) { var pos = game.player.position; var dir = game.player.forward; move(pos.x + dir.x * 1.5, pos.y, pos.z + dir.z * 1.5); } }); }

4. WallHack & ESP (Подсветка врагов)

Сканирует массив всех NPC/Монстров на карте и отрисовывает 2D маркеры с их HP поверх экрана.
game.every(0.2, function() { game.ui.clear(); if (typeof _npcs !== 'undefined' && _npcs.length > 0) { _npcs.forEach(function(npc) { if (npc.hp > 0) { game.ui.set("wh_" + npc.id, "🔴 " + npc.name + " [" + Math.round(npc.hp) + " HP]", { x: npc.x, y: npc.y + 2, z: npc.z, color: "#ff0055", fontSize: "14px" }); } }); } });
